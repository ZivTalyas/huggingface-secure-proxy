Validate via API

Backend Endpoint: POST /validate

1. Validate Text (JSON)
{
  "text": "Hello, please review this message.",
  "security_level": "high"
}
2. Validate File (multipart/form-data)
curl -X POST http://localhost:8001/validate \
  -F "file=@example.pdf" \
  -F "security_level=high"

Frontend Endpoint: POST /validate-input

a. Validate Text (JSON)
{
  "text": "Hello from frontend!",
  "security_level": "medium"
}
b. Validate File (multipart/form-data)
curl -X POST http://localhost:8000/validate-input \
  -F "file=@example.txt" \
  -F "security_level=medium"
